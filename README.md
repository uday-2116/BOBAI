# BOBAI
A speaking robot companion for kids below 10 years. it can say stories like Ramayana and Mahabharatha, play songs and rhymes, crack jokes and helps them in education and gain knowledge. it can controlled by a controller, So that it will be like a playful toy to them, where they can learn something new
